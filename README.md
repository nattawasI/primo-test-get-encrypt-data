# Primo Encryption API

A NestJS project for encrypting and decrypting data using AES and RSA.

## Requirements
- Node.js
- NestJS CLI

## Setup
```bash
npm install
```

## Running the app
```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Documentation
Once running, you can access the Swagger documentation at:
`http://localhost:3000/api-docs`
